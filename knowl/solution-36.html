Solution 2.2.13.1.

Parametrize \(\mathcal{C}\) as \(\boldr(t)=(\cos t, \sin t)\text{,}\) \(0\leq t\leq 2\pi\text{.}\) It follows that \(\boldr'(t)=\angvec{-\sin t, \cos t}\text{,}\) and hence that the outward unit normal vector is \(\boldn(t)=\angvec{\cos t, \sin t}\text{.}\) We now compute the flux in each case using DefinitionÂ 2.2.12.

  1. \begin{align*} \int_\mathcal{C}\boldF\cdot \boldn\, ds \amp= \int_0^{2\pi}\boldF(\cos t, \sin t)\cdot \boldn(t)\, ds \\ \amp = \int_0^{2\pi}\angvec{\cos t-\sin t, \cos t+\sin t}\cdot \angvec{\cos t, \sin t}\, dt\\ \amp = \int_0^{2\pi} \cos^2 t-\cos t\sin t+\cos t\sin t+\sin^2 t\, dt\\ \amp = \int_0^{2\pi}1\, dt\\ \amp = 2\pi\text{.} \end{align*}

  2. \begin{align*} \int_\mathcal{C}\boldF\cdot \boldn\, ds \amp= \int_0^{2\pi}\boldF(\cos t, \sin t)\cdot \boldn(t)\, ds \\ \amp = \int_0^{2\pi}\angvec{\cos t, \sin t}\cdot \angvec{\cos t, \sin t}\, dt\\ \amp = \int_0^{2\pi} 1 dt\\ \amp = 2\pi\text{.} \end{align*}
