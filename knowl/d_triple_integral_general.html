Definition 1.5.3. Triple integral over bounded region.

Let \(f\) be a function of three variables defined over the bounded region \(\mathcal{D}\subseteq\R^3\text{.}\) Let \(B\) be any box containing \(\mathcal{D}\) and let \(f^*\) be the function defined on the box \(B\) as follows:

\begin{equation*} f^*(x,y,z)=\begin{cases}f(x,y,z)\amp \text{if } (x,y,z)\in \mathcal{D}\\ 0\amp \text{if } (x,y,z)\notin \mathcal{D} \end{cases}\text{.} \end{equation*}

The function \(f\) is integrable over \(\mathcal{D}\) if the function \(f^*\) is integrable over the box \(B\text{,}\) and in this case we define

\begin{equation*} \iiint\limits_\mathcal{D}f(x,y,z)\, dV=\iiint\limits_{B}f^*(x,y,z)\, dV\text{.} \end{equation*}
