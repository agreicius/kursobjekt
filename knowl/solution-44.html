Solution 2.6.14.1.

We use the usual parametrization of the sphere of radius 2:

\begin{align*} \boldr(\phi,\theta) \amp = (2\sin\phi\cos\theta)\\ \amp \end{align*}

Given a point \(P=(x,y,z)\in \mathcal{S}\) the vector \(\angvec{x,y,z}\) is an outward pointing normal vector of magnitude \(\sqrt{x^2+y^2+z^2}=2\text{.}\) Thus the function \(\boldn(x,y,z)=\frac{1}{2}\angvec{x,y,z}\) is the outward orientation on \(\mathcal{S}\text{.}\) It follows that

\begin{equation*} \boldF(x,y,z)\cdot \boldn(x,y,z)=\frac{1}{2}\angvec{z,y,x}\cdot \angvec{x,y,z}=\frac{1}{2}(2xz+y^2)\text{,} \end{equation*}

and thus that

\begin{align*} \iint\limits_\mathcal{S}\boldF\cdot\boldn\, d\sigma\amp = \frac{1}{2}\iint\limits_\mathcal{S}2xz+y^2\, d\sigma \\ \amp = \frac{1}{2}\iint\limits_{\mathcal{R}}(8\sin\phi\cos\phi\cos\theta+4\sin^2\phi\sin^2\theta)(4\sin\phi)\, dA\\ \amp =8\int_0^{2\pi}\int_0^\pi2\sin^2\phi\cos\phi\cos\theta+\sin^3\phi\sin^2\theta\, d\phi\, d\theta\\\ \amp = 8\left(\cancelto{0}{\int_0^{2\pi}\cos\theta\, d\theta}\int_0^{\pi}\sin^2\phi\cos\phi\, d\phi+\int_0^{2\pi}\sin^2\theta\, d\theta\int_0^{\pi}\sin^3\phi\, d\phi\right)\\ \amp =8\int_0^{2\pi}\sin^2\theta\, d\theta\int_0^{\pi}\sin^3\phi\, d\phi\\ \amp =\frac{32\pi}{3}\text{.} \end{align*}

The last step above is done using the identity \(\sin^2\theta=\frac{1}{2}(1-\cos 2\theta)\) and the fact that \(\sin^3\phi=\sin\phi(1-\cos^2\phi)=\sin\phi-\sin\phi\cos^2\phi\) has antiderivative \(-\cos\phi+\frac{1}{3}\cos^3\phi\text{.}\)
