Corollary 2.3.4. Integral of gradient field over closed curve.

If \(\mathcal{C}\subseteq\R^n\) is a smooth closed curve, and if \(\boldF=\nabla f\) is a gradient vector field that is continuous on a set containing \(\mathcal{C}\text{,}\) then

\begin{equation*} \oint_\mathcal{C}\boldF\cdot\, d\boldr=0\text{.} \end{equation*}
