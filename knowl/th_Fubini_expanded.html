Theorem 1.5.9. Fubini's theorem expanded.

Assume \(f\) is continuous on the elementary region \(\mathcal{D}\subseteq\R^3\text{.}\)

  • Type A1.

    If \(\mathcal{D}=\{(x,y,z)\in \R^3, a\leq x\leq b, g_1(x)\leq y\leq g_2(x), p_1(x,y)\leq z\leq p_2(x,y)\}\text{,}\) then

    \begin{equation} \iiint\limits_\mathcal{D}f(x,y,z)=\int_a^b\int_{g_1(x)}^{g_2(x)}\int_{p_1(x,y)}^{p_2(x,y)}f(x,y,z)\, dz\, dy\, dx.\tag{1.5.7} \end{equation}
  • Type A2.

    If \(\mathcal{D}=\{(x,y,z)\in \R^3, c\leq y\leq d, h_1(y)\leq x\leq h_2(y), p_1(x,y)\leq z\leq p_2(x,y)\}\text{,}\) then

    \begin{equation} \iiint\limits_\mathcal{D}f(x,y,z)=\int_c^d\int_{h_1(y)}^{h_2(y)}\int_{p_1(x,y)}^{p_2(x,y)}f(x,y,z)\, dz\, dx\, dy.\tag{1.5.8} \end{equation}
  • Type B1.

    If \(\mathcal{D}=\{(x,y,z)\in \R^3, a\leq y\leq b, g_1(y)\leq z\leq g_2(y),q_1(y,z)\leq x\leq q_2(y,z)\}\text{,}\) then

    \begin{equation} \iiint\limits_\mathcal{D}f(x,y,z)=\int_a^b\int_{g_1(y)}^{g_2(y)}\int_{p_1(y,z)}^{p_2(y,z)}f(x,y,z)\, dx\, dz\, dy.\tag{1.5.9} \end{equation}
  • Type B2.

    If \(\mathcal{D}=\{(x,y,z)\in \R^3, c\leq z\leq d, h_1(z)\leq y\leq h_2(z), q_1(y,z)\leq x\leq q_2(y,z)\}\text{,}\) then

    \begin{equation} \iiint\limits_\mathcal{D}f(x,y,z)=\int_c^d\int_{h_1(z)}^{h_2(z)}\int_{p_1(y,z)}^{p_2(y,z)}f(x,y,z)\, dx\, dy\, dz.\tag{1.5.10} \end{equation}
  • Type C1.

    If \(\mathcal{D}=\{(x,y,z)\in \R^3, a\leq x\leq b, g_1(x)\leq z\leq g_2(x), r_1(x,z)\leq y\leq r_2(y,z)\}\text{,}\) then

    \begin{equation} \iiint\limits_\mathcal{D}f(x,y,z)=\int_a^b\int_{g_1(x)}^{g_2(x)}\int_{p_1(y,z)}^{p_2(y,z)}f(x,y,z)\, dx\, dz\, dy.\tag{1.5.11} \end{equation}
  • Type C2.

    If \(\mathcal{R}=\{(x,y,z)\in \R^3, c\leq z\leq d, h_1(z)\leq x\leq h_2(z), r_1(x,z)\leq y\leq r_2(y,z)\}\text{,}\) then

    \begin{equation} \iiint\limits_\mathcal{D}f(x,y,z)=\int_c^d\int_{h_1(z)}^{h_2(z)}\int_{p_1(y,z)}^{p_2(y,z)}f(x,y,z)\, dx\, dy\, dz.\tag{1.5.12} \end{equation}
