Theorem 2.3.13.

Let \(\boldF=\angvec{F_1,F_2,F_3}\text{,}\) and assume the component functions have continuous partial derivatives on the domain of \(\boldF\text{.}\) If \(\boldF=\nabla f\) for some scalar function \(f\text{,}\) then \(\curl \boldF=\nabla\times \boldF=\boldzero\text{.}\) Using logical shorthand:

\begin{equation} \boldF=\nabla f\implies \nabla\times \boldF=\boldzero\text{.}\tag{2.3.7} \end{equation}
