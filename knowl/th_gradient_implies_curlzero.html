Theorem 2.3.13. Gradient implies zero curl.

Let \(\boldF\colon \R^3\rightarrow \R^3\) be a vector field whose component functions have continuous partial derivatives on the set \(D\text{.}\) If \(\boldF=\nabla f\) for some scalar function \(f\text{,}\) then \(\curl \boldF(x,y,z)=\nabla\times \boldF(x,y,z)=\angvec{0,0,0}\) for all \((x,y,z)\in D\) . Using logical shorthand:

\begin{equation} \boldF=\nabla f\implies \nabla\times \boldF=\boldzero\text{.}\tag{2.3.7} \end{equation}
