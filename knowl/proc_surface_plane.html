Procedure 2.5.3. Plane parametrization.

Recall that a plane \(\mathcal{S}\) is defined (implicitly) as the set of solutions to an equation of the form

\begin{equation} \mathcal{S}\colon ax+by+cz=d\text{,}\tag{2.5.1} \end{equation}

where \(\boldn=\angvec{a,b,c}\ne \angvec{0,0,0}\text{,}\) is called a normal vector of the plane. More geometrically, given a particular point \(P=(x_0,y_0,z_0)\) satisfying (2.5.1), we can describe \(\mathcal{P}\) as the set of points \(Q=(x,y,z)\) for which the vector \(\overrightarrow{PQ}=\angvec{x-x_0,y-y_0,z-z_0}\) is orthogonal to \(\boldn\text{,}\) which we call a normal vector to the plane.

To parametrize a plane \(\mathcal{S}\) defined by (2.5.1) proceed as follows.

  1. Find a point \(P=(x_0,y_0,z_0)\) on \(\mathcal{S}\text{:}\) i.e., find one solution to (2.5.1).

  2. Pick two nonzero vectors \(\boldw_1, \boldw_2\) that are orthogonal to \(\boldn\) and are not scalar multiples of one another: i.e.,

    \begin{equation*} \boldn\cdot \boldw_1=\boldn\cdot \boldw_2=0, \boldv_1\ne c\boldv_2\text{.} \end{equation*}

  3. The plane \(\mathcal{S}\) is the set of points \(Q\) of the form

    \begin{equation*} Q=P+u\boldw_1+v\boldw_2\text{,} \end{equation*}

    for some choice of scalars \(u,v\in \R\text{.}\)

  4. If \(\boldw_1=\angvec{r_1,r_2,r_3}\) and \(\boldw_2=\angvec{s_1,s_2,s_3}\text{,}\) then the function \(\boldr\colon \R^2\rightarrow \R^3\) defined as

    \begin{equation*} \boldr(u,v)=(x_0,y_0,z_0)+u\boldw_1+v\boldw_2=(x_0+ur_1+vs_1,y_0+ur_2+vs_2,z_0+ur_3+vs_3) \end{equation*}

    is a parametrization of the plane \(\mathcal{S}\text{.}\)
