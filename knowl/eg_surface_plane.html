Example 2.5.3. Plane parametrization.

Recall that a plane \(\mathcal{S}\) is defined (implicitly) as the set of solutions to an equation of the form

\begin{equation} \mathcal{S}\colon ax+by+cz=d\text{,}\tag{2.5.1} \end{equation}

where \(\boldn=\angvec{a,b,c}\ne \angvec{0,0,0}\text{.}\) Alternatively, given a particular point \(P=(x_0,y_0,z_0)\) satisfying (2.5.1), we can describe \(\mathcal{P}\) as the set of points \(Q=(x,y,z)\) for which the vector \(\overrightarrow{PQ}=\angvec{x-x_0,y-y_0,z-z_0}\) is orthogonal to \(\boldn\text{.}\) Let \(\mathcal{P}\) be the plane through the point \(P=(x_0,y_0,z_0)\) with normal vector \(\boldn=\angvec{a,b,c}\text{.}\)
