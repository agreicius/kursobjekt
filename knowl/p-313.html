Geometrically, we can think of a tuple either as a point \(P\) in \(\R^n\text{,}\) or as an \(n\)-vector \(\boldv=\langle v_1,v_2,\dots, v_n\rangle\text{.}\) When visualizing a path \(\boldr\) and its outputs, we will make use of both interpretations. The table below recalls some standard interpretations of \(\boldr(t)\) and \(\boldr'(t)\) along these lines. For simplicity, we assume \(n=3\text{.}\)

\begin{equation*} \begin{array}{c|c|c} \amp \text{As point} \amp \text{As vector}\\ \hline \boldr(t)=(a,b,c) \amp \begin{array}{l} \text{point } P=(a,b,c)\\ \text{on curve } \mathcal{C} \end{array} \amp \begin{array}{l} \text{position vector } \overrightarrow{OP}=\angvec{a,b,c}\\ \text{of point } P\in \mathcal{C} \end{array} \\ \hline \boldr'(t)=(d,e,f)\amp ---\amp \begin{array}{l} \text{tangent vector } \boldv=\angvec{d,e,f} \\ \text{to curve } \mathcal{C} \text{ at point } \boldr(t)=P. \end{array} \end{array} \end{equation*}
