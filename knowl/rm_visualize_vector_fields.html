Remark 2.2.2. Visualizing vector fields.

Like a curve parametrization, a vector field \(\boldF\colon \R^n\rightarrow \R^n\) is yet another example of a vector-valued function. In contrast to parametrizations, however, note that both the input and output of \(\boldF\) are \(n\)-tuples. This makes visualizing \(F\) slightly more complicated. We use the following method to do so. As always, \(n\) is equal to 2 or 3 in our course.

  1. Interpret the input tuple \(P=(x_1,x_2,\dots, x_n)\) as a point in \(n\)-space.

  2. Treat the output tuple \(\boldv=\boldF(P)=\langle F_1(P),F_2(P),\dots, F_n(P)\rangle \) as an \(n\)-vector.

  3. For a given input \(P\in \R^n\text{,}\) draw the output \(\boldv=\boldF(P)\) as a vector with initial point equal to \(P\text{:}\) i.e., “tail” of the vector \(\boldF(P)\) is chosen to be the point \(P\text{.}\)

  4. Sketch the entire vector field by performing Step 3 for many points!
