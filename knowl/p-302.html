Let \(\mathcal{R}\subseteq\R^2\) have area \(A\text{.}\) The random variables \(X, Y\) are uniformly distributed over \(\mathcal{R}\) if their joint probabiltity disribution function is given by

\begin{equation*} f(x,y)=\begin{cases} \frac{1}{A}\amp \text{if } (x,y)\in \mathcal{R}\\ 0 \amp \text{if } (x,y)\not\in \mathcal{R} \end{cases}. \end{equation*}
