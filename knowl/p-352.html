Let \(\boldr\colon [a,b]\rightarrow \R^n\) be a smooth parametrization of the curve \(\mathcal{C}\subseteq\R^n\) with initial point \(P=\boldr(a)\) and end point \(Q=\boldr(b)\text{,}\) and assume the vector field \(\boldF\colon \R^n\rightarrow \R^n\) is continuous on a set containing \(\mathcal{C}\text{.}\) If \(\boldF=\nabla f\) for some function \(f\text{,}\) then

\begin{equation} \int_\mathcal{C}\boldF\cdot d\boldr=f(Q)-f(P)\text{.}\tag{2.3.2} \end{equation}
