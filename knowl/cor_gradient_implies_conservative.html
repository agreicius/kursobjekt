Corollary 2.3.8. Gradient fields are conservative.

If \(\boldF=\nabla f\) is a continuous gradient vector field on the open set \(D\text{,}\) then \(\boldF\) is conservative on \(D\text{.}\) Using logical shorthand:

\begin{equation} \boldF=\nabla f \implies \boldF \text{ conservative}\text{.}\tag{2.3.4} \end{equation}

Proof.
