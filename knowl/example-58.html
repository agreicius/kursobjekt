Example 2.6.10. Orientations of the sphere.

Let \(\mathcal{S}\) be the unit sphere \(x^2+y^2+z^2=1\text{.}\) We have seen previously that for a point \(P=(x,y,z)\) on \(\mathcal{S}\) the vector \(\overrightarrow{OP}=\angvec{x,y,z}\) is a normal vector to \(\mathcal{S}\) at \(P\text{.}\) Since the function \(\boldn_1(x,y,z)=\angvec{x,y,z}\) is clearly continuous, it defines an orientation on \(\mathcal{S}\text{,}\) called an outward orientation. The function \(\boldn_2(x,y,z)=-\angvec{x,y,z}\) also defines an orientation on the sphere, called the inward orientation.
