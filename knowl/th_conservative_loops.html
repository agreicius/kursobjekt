Theorem 2.3.8. Loop property of conservative fields.

If \(\boldF\) is a continuous vector field on the open set \(D\text{,}\) then \(\boldF\) is conservative if and only if \(\oint_\mathcal{C}\boldF\cdot d\boldr=0\) for all piecewise smooth closed curves \(\mathcal{C}\subseteq D\text{.}\) Using logical shorthand:

\begin{equation} \boldF \ \text{ conservative } \iff \oint_\mathcal{C}\boldF\cdot d\boldr=0\text{.}\tag{2.3.4} \end{equation}

Proof: conservative \(\ \implies \oint_\mathcal{C}\boldF\cdot d\boldr=0\).

Proof: \(\oint_\mathcal{C}\boldF\cdot d\boldr=0 \implies\) conservative.
