Theorem 2.3.10. Conservative field=gradient field on open connected sets.

If \(\boldF\) is a continuous vector field on the open connected set \(D\text{,}\) then \(\boldF\) is conservative if and only if \(\boldF=\nabla f\) for some function \(f\) defined on \(D\text{.}\) Using logical shorthand:

\begin{equation} \boldF=\nabla f \iff \boldF \text{ conservative}\text{.}\tag{2.3.5} \end{equation}

Proof of implication \(\boldF=\nabla f\implies \boldF\) conservative.

Proof of implication \(\boldF\) conservative \(\implies \boldF=\nabla f\).
