Let \(\boldF(x,y)=\angvec{F_1(x,y), F_2(x,y)}\) be a vector field whose component functions have continuous partial derivatives on the set \(D\text{.}\) If \(\boldF=\nabla f\) for some scalar function, then \(\partial F_2/\partial x(x,y)-\partial F_1/\partial y(x,y)=0\) for all \((x,y)\in D\text{.}\)
